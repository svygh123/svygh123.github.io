一站式缴费系统

发布日期：2024-08-27 作者：svygh123 分类：java
springboot+thymeleaf+MyBatis+MySQL三层架构开发的一站式缴费系统，html 5、bootstrap响应式技术开发，echarts可视化图表，稳定运行，代码规范。

一站式缴费系统说明文档

1. 前台技术

  • html 5
  • bootstrap v3.3.7
  • echarts 4.3.1

2. 后台技术

  • springboot 3.1.3
  • MySQL 8
  • thymeleaf-3.1.2
  • tomcat-10.1.12
  • maven 3.9.6

3. 主要功能

  • 用户信息
  • 账户信息
  • 服务类型信息
  • 户号信息
  • 账单信息
  • 支付记录信息
  • 日志信息
  • 个人信息

4. 系统特性

  • B/S架构
  • MVC模式
  • sql防注入

5. 安装步骤

打开ide(eclipse/idea/vscode等)    
导入项目
安装数据库(MySQL 8+)
运行paysys.sql数据库脚本
项目中配置的数据库默认信息
数据库：paysys  
用户名：root    
密码：123456    
如果本机数据库和默认不一致，需要修改配置 
application-dev.yml
