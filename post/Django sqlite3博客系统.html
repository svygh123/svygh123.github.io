Django sqlite3博客系统

发布日期：2024-08-24 作者：svygh123 分类：python
Django+sqlite3三层架构开发的博客系统，html 5、bootstrap-3.4.1响应式开发。

Django+sqlite3博客系统说明文档

1. 前台技术

  • html 5
  • bootstrap 3.4.1

2. 后台技术

  • python 3.8
  • sqlite3 数据库
  • Django 4.2.11

3. 主要功能

  • 注册、登录
  • 发布文章
  • 文章管理
  • 文章分类管理
  • 文章按分类查看
  • 文章标签管理
  • 文章按标签查看
  • 文章归档
  • 热门文章

4. 系统特性

  • MVC三层架构
  • sql防注入
  • 标签词云

5. 安装步骤

    下载安装python 3.8及以上
    安装Django(Django 4.2.11)
    打开ide(vscode/idea/eclipse等)
    导入项目

6. 运行

python manage.py runserver 0.0.0.0:8000

7. 访问地址

  • http://localhost:8000/blog
  • 账号：admin 密码：asdfgh123456

8. 开发环境

  • Windows 7
  • Visual Studio Code 1.70.2
  • sqlite3
