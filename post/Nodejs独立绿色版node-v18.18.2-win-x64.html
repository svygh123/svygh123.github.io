Nodejs独立绿色版node-v18.18.2-win-x64

发布日期：2024-08-22 作者：svygh123 分类：nodejs
Nodejs独立绿色版，Nodejs一键运行包，Nodejs一键启动包，Nodejs便携版

1.建议解压到除C盘以外的目录

2.双击运行文件：一键运行.bat，弹出命令行窗口

3.然后就可以在弹出的命令行窗口里面输入命令了，在这里cd切换到哪个目录都可以运行

4.测试node环境命令如下

  • 查看nodejs版本：node -v
  • 查看npm版本：npm -v
  • 查看node配置：npm config list

5.创建hello.js运行

输入

echo console.log("Hello,NodeJS.")>hello.js

回车

输入

node hello.js

回车

就会在屏幕打印出来：

Hello,NodeJS.

Windows 7及以上版本都可以使用
