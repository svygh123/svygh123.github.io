学生信息管理系统

发布日期：2024-08-19 作者：svygh123 分类：java
JSP+Servlet+MySQL三层架构开发的学生信息管理系统，html 5、bootstrap 5最新技术开发，echarts可视化图表，稳定运行，代码规范。

学生信息管理系统说明文档

1. 前台技术

  • html 5
  • bootstrap 5.3.3
  • echarts 4.3.1

2. 后台技术

  • servlet 3.1
  • jdbc 连接数据库
  • MySQL 8
  • jstl-1.2
  • JSP

3. 主要功能

  • 管理员信息
  • 公告信息:带富文本
  • 部门信息
  • 专业信息
  • 班级信息
  • 教师信息
  • 课程信息
  • 学生信息
  • 作业信息:带附件
  • 作业提交:带附件
  • 成绩信息

4. 系统特性

  • B/S架构
  • MVC模式
  • sql防注入
  • 实现自动登录

5. 安装步骤

打开ide(eclipse/idea/vscode等)，下载安装jdk,tomcat8    
导入项目
安装数据库(8+)
运行stuinfosys.sql数据库脚本
项目中配置的数据库默认信息
数据库：stuinfosys    
用户名：root    
密码：123456    
如果本机数据库和默认不一致，需要修改，路径如下    
stuinfosys/src/com/stu/util/DbUtil.java
